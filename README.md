# Lista de Tareas con Node + Express

Este proyecto implementa la funcionalidad del CRUD  de una lista de tareas y Login.

# Descargar el proyecto y correlo localmente

Si desea correr el proyecto localmente realice los siguiente. 
1. Descargue o clone el proyecto 
2. En la carpeta raiz ejecute npm install para instalar todas las dependencias necesesarias 
3. Ejecute localmente el proyecto usando el comando npm run dev
4. Puede probar los enpoint haciendo uso del archivo requestTask.http, requestUser.http  o postman o
5. Tambien puede descargar o clonar el proyecto Angular https://github.com/rivaldoa10/firebase-task-list-with-login-angular-17
